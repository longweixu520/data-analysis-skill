# 数据分析方法资源库

面向**政务统计、区域经济、企业经营、行业对标**等表格型数据分析任务的方法索引。

## 使用方式

- **数据侦查手**：01、02
- **数据工程手**：01、02
- **分析计算手**：03、04、05、06
- **交付汇报手**：07

按需加载，不要一次性读完全部。

---

## 文档目录

| 序号 | 文档 | 核心内容 |
|------|------|----------|
| 01 | 数据解析与结构识别 | 11 种表结构、解析决策树、pandas 技巧 |
| 02 | 数据清洗与质量治理 | 缺失、异常、口径、审计 |
| 03 | 描述统计与对比分析 | 排名、同比环比、分组对比 |
| 04 | 综合评价与权重方法 | 熵值法、AHP、TOPSIS、CRITIC |
| 05 | 时间序列与趋势分析 | 趋势、CAGR、断点、季节性 |
| 06 | 统计推断与假设检验 | 相关、回归、t 检验、效应量 |
| 07 | 可视化与报告规范 | Figure Contract、中文图表 |

---

## 方法选择原则

```
1. 描述能答不问推断
2. 截面排名不问因果
3. 综合指数不替代单指标解读
4. 图表数量 ≤ 子问题数量 + 1
```

---

## 与数学建模 skill 的边界

| 本 skill | math-modeling-skill |
|----------|---------------------|
| 真实业务表格、统计年鉴 | 竞赛题目、数学模型 |
| 长表 + 排名 + 综合指数 | 优化/预测/仿真模型 |
| 中文交付报告 | 学术论文 |

二者可组合：复杂预测/优化问题可在分析计算阶段引用 math-modeling 的算法资源。
