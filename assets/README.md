## 方法选择原则

```
1. 描述能答不问推断
2. 截面排名不问因果
3. 综合指数不替代单指标解读
4. 图表数量 ≤ 子问题数量 + 1
5. 预测仅在用户明确要求且数据长度足够时进行
```

## 适用范围

| 适合 | 不适合（需专项方法或外部数据） |
|------|-------------------------------|
| 统计年鉴、政府公开表、企业经营表 | 实时流数据、超大规模日志 |
| 城市/企业/地区横向对比 | 需要严格因果推断的实验设计 |
| 综合指数、排名、同比环比 | 深度学习端到端预测（非本 skill 主路径） |
| 中文 Markdown/Word 交付 | 交互式 BI 看板搭建 |

复杂预测需求可在本 skill 框架内使用 05 中的轻量方法，并在报告中声明不确定性。
