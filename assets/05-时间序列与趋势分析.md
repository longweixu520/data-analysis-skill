## 断点识别

统计口径变更常表现为某年全员跳变：

```python
def detect_jump(series, threshold=0.5):
    pct = series.pct_change()
    return pct[abs(pct) > threshold].index.tolist()
```

发现断点后**分段分析**，不跨段算同比。

## 趋势表述

| 数据表现 | 表述 |
|----------|------|
| 单调上升 | 「2018–2024 年持续上升，年均 +X%」 |
| 先升后降 | 「2021 年达峰后回落」 |
| 波动 | 「呈波动上行，无单一趋势」 |

避免在无模型支撑时写「未来将…」。

## 轻量预测（仅在用户明确要求时）

| 数据长度 | 推荐方法 | 注意 |
|----------|----------|------|
| < 5 年 | 移动平均、线性外推 | 仅作参考，置信区间宽 |
| 5–15 年 | Holt 线性趋势、ARIMA(1,1,1) | 需检验残差 |
| 高波动短序列 | 灰色预测 GM(1,1) | 适合小样本单调趋势 |

```python
# 线性趋势外推示例（谨慎使用）
import numpy as np
years = np.array([2019, 2020, 2021, 2022, 2023])
values = np.array([...])
coef = np.polyfit(years, values, 1)
forecast_2024 = np.polyval(coef, 2024)
```
