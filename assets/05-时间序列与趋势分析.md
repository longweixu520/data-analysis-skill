## 断点识别

统计口径变更常表现为某年全员跳变：

```python
def detect_jump(series, threshold=0.5):
    pct = series.pct_change()
    return pct[abs(pct) > threshold].index.tolist()
```

发现断点后**分段分析**，不跨段算同比。

## 趋势表述

| 数据表现 | 表述 |
|----------|------|
| 单调上升 | 「2018–2024 年持续上升，年均 +X%」 |
| 先升后降 | 「2021 年达峰后回落」 |
| 波动 | 「呈波动上行，无单一趋势」 |

避免在无模型支撑时写「未来将…」。

---

## 与预测模型边界

若用户明确要求预测：

- 短序列（<5 年）：移动平均、线性趋势
- 中长序列：引用 math-modeling-skill 的 ARIMA/灰色预测
- 必须在报告中声明预测不确定性与数据长度限制
