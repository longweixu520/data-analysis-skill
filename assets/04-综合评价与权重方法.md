# 04 - 综合评价与权重方法

## 方法对比

| 方法 | 类型 | 优点 | 缺点 |
|------|------|------|------|
| 等权 | 客观 | 简单透明 | 忽略指标差异 |
| 熵值法 | 客观 | 区分度驱动 | 对数据分布敏感 |
| CRITIC | 客观 | 考虑相关性与对比强度 | 计算稍复杂 |
| AHP | 主观 | 体现政策导向 | 需专家判断 |
| TOPSIS | 排序 | 贴近理想解 | 需先标准化+赋权 |
| 组合赋权 | 混合 | 主客观平衡 | 需约定组合系数 |

**默认：熵值法**。用户强调政策导向时用 AHP 或组合赋权。

## 熵值法完整实现

```python
import numpy as np
import pandas as pd

def entropy_weights(norm_matrix: pd.DataFrame) -> pd.Series:
    """norm_matrix: 行=主体, 列=指标, 值∈[0,1]"""
    X = norm_matrix.values.astype(float)
    m, n = X.shape
    if m < 2:
        return pd.Series(1 / n, index=norm_matrix.columns)
    # 平移防 0
    X = X + 1e-12
    P = X / X.sum(axis=0)
    k = 1 / np.log(m)
    E = -k * (P * np.log(P)).sum(axis=0)
    D = 1 - E
    W = D / D.sum()
    return pd.Series(W, index=norm_matrix.columns)
```

## 截面标准化（按年）

```python
def normalize_by_year(df, indicators, directions):
    """directions: dict 指标名 -> True(正向)"""
    rows = []
    for year, g in df.groupby("年份"):
        pivot = g.pivot(index="主体", columns="指标名称", values="清洗值")
        norm = pd.DataFrame(index=pivot.index)
        for ind in indicators:
            s = pivot[ind]
            pos = directions.get(ind, True)
            lo, hi = s.min(), s.max()
            if hi == lo:
                norm[ind] = 1.0
            elif pos:
                norm[ind] = (s - lo) / (hi - lo)
            else:
                norm[ind] = (hi - s) / (hi - lo)
        norm["年份"] = year
        rows.append(norm.reset_index())
    return pd.concat(rows, ignore_index=True)
```

## TOPSIS（可选）

1. 标准化矩阵 Z
2. 加权 `V = Z * W`
3. 正理想解 A+、负理想解 A-
4. 距离 D+, D-
5. 贴近度 `C = D- / (D+ + D-)`

适用于「找最接近标杆」场景。

## AHP 要点

成对比较矩阵 → 特征向量得权重 → 一致性比率 CR < 0.1。

主观权重与熵值法组合：

```
w_final = α * w_ahp + (1-α) * w_entropy
```

α 默认 0.5，用户可指定。
