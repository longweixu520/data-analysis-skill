## 最新年份排名

```python
def latest_ranking(df, value_col="清洗值", score_name="指标值"):
    latest_year = df["年份"].max()
    sub = df[df["年份"] == latest_year].copy()
    sub = sub.sort_values(value_col, ascending=False)
    sub["排名"] = range(1, len(sub) + 1)
    return sub[["主体", "年份", score_name, "排名"]]
```

## 同比 / 环比

```python
def add_yoy(df, group_cols=["主体", "指标名称"], value_col="清洗值"):
    df = df.sort_values(group_cols + ["年份"])
    g = df.groupby(group_cols)
    df["上期"] = g[value_col].shift(1)
    df["同比"] = (df[value_col] - df["上期"]) / df["上期"]
    df.loc[df["上期"].isin([0, None]) | df["上期"].isna(), "同比"] = pd.NA
    return df
```

## 区间增幅与 CAGR

```python
def cagr(start, end, n_years):
    if start <= 0 or end <= 0 or n_years <= 0:
        return pd.NA
    return (end / start) ** (1 / n_years) - 1
```

## 分组对比

| 分析 | 方法 |
|------|------|
| 区域对比 | groupby 区域 |
| 结构占比 | 年内各主体值 / 总和 |
| 贡献分解 | 增量 = 本期 - 上期，占比 = 增量 / 总增量 |

## 水平 vs 增速

- **水平**：最新年排名、多年均值排名
- **增速**：同比、CAGR、区间增幅

二者可能背离，报告必须分开写。
