## 线性回归（探索）

```python
import statsmodels.api as sm

X = sm.add_constant(df["人均GDP"])
y = df["研发投入"]
model = sm.OLS(y, X).fit()
# 报告：R², 系数, p-value, 样本量 n
```

## t 检验（两组均值）

```python
from scipy import stats
t, p = stats.ttest_ind(group_a, group_b, nan_policy="omit")
```

同时报告效应量（Cohen's d）：

```python
def cohens_d(a, b):
    na, nb = len(a), len(b)
    pooled_std = np.sqrt(((na-1)*a.std()**2 + (nb-1)*b.std()**2) / (na+nb-2))
    return (a.mean() - b.mean()) / pooled_std
```

---

## 报告规范

- 报告 `n`、检验方法、p 值、效应量
- p > 0.05 写「差异不显著」，不写「无差异」
- 小样本（n<30）谨慎推断
