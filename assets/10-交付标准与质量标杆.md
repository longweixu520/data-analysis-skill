# 10 - 交付标准与质量标杆

优秀交付 = **可复现 + 可审计 + 可读 + 可展示**。

---

## 一级交付（最低合格）

- [ ] `02_清洗后长表.csv` schema 正确
- [ ] `03_数据质量检查表.csv` 已生成（`quality_report.py`）
- [ ] 分析结果表与结论数字一致
- [ ] `validate_delivery.py` 通过

## 二级交付（推荐）

- [ ] 排名 + 同比 + 综合指数（若适用）
- [ ] 图表 ≥2 张，中文标注，PNG+SVG
- [ ] `数据分析报告.md` 含摘要、方法、风险
- [ ] `src/run_pipeline.py` 可一键重跑

## 三级交付（标杆）

- [ ] `数据分析报告.html` 单文件可分享
- [ ] `数据分析汇总.xlsx` 多 sheet 汇总
- [ ] Data Contract + 术语表完整
- [ ] 每张图有 Figure Contract 解读（≥100 字）
- [ ] 补值/口径假设在报告「风险」章节明示

## 性能建议

| 数据规模 | 做法 |
|----------|------|
| < 1 万行 | pandas 全流程 |
| 1–50 万行 | polars 读入 + pandas 分析 |
| > 50 万行 | DuckDB SQL + 抽样图表 |
