# 01 - 数据解析与结构识别

## 概述

统计年鉴、行业报表、政府公开数据的难点不在计算，而在**表头语义识别**。本指南提供可复用的解析模式与代码骨架。

| 格式 | 首选库 | 注意 |
|------|--------|------|
| .xlsx | pandas + openpyxl | `header=None` 先读再判 |
| .xls | pandas + xlrd | 老格式 |
| .csv | pandas / polars | 先检测编码 |

## 探测代码模板

```python
import pandas as pd

def probe_excel(path: str, nrows: int = 15) -> dict:
    xl = pd.ExcelFile(path)
    result = {"sheets": xl.sheet_names, "samples": {}}
    for name in xl.sheet_names:
        df = pd.read_excel(path, sheet_name=name, header=None, nrows=nrows)
        result["samples"][name] = df
    return result
```

## melt 示例（模式 A）

```python
def matrix_to_long(df, subject_col, year_cols, indicator_name, source_meta):
    long_df = df.melt(
        id_vars=[subject_col],
        value_vars=year_cols,
        var_name="年份",
        value_name="原始值",
    )
    long_df["指标名称"] = indicator_name
    long_df["年份"] = long_df["年份"].astype(int)
    for k, v in source_meta.items():
        long_df[k] = v
    return long_df
```

## 年份列识别

```python
import re

def is_year_col(col) -> bool:
    s = str(col).strip()
    if re.fullmatch(r"20\d{2}", s):
        return True
    if re.fullmatch(r"20\d{2}年", s):
        return True
    return False
```

## 常见陷阱

1. **合并单元格**：openpyxl 读入后 forward-fill 主体列
2. **数字存为文本**：`pd.to_numeric(errors="coerce")`
3. **全角字符**：`str.normalize` + 替换全角数字
4. **隐藏行/列**：用 openpyxl 检查 `sheet.max_row`
5. **多个表在一个 sheet**：用空行比例切分
