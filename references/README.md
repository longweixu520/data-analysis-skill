# 四角色协作指南

本目录包含数据分析流水线四个阶段的角色子 Skill。

## 流水线

```
数据侦查手 → 数据工程手 → 分析计算手 → 交付汇报手
     ↓            ↓            ↓            ↓
 侦查报告      标准长表      分析结果表      报告+图表
 Data Contract  质量检查表    权重/排名      交付清单
```

## 渐进式加载

每个阶段**仅加载当前角色 SKILL.md**，详细引用按需打开 `references/` 下文件。

## 阶段门禁

| 从 | 到 | 门禁条件 |
|----|-----|----------|
| 侦查 | 工程 | Data Contract 完成 + 侦查质检通过 |
| 工程 | 计算 | 02_清洗后长表.csv 存在 + 工程质检通过 |
| 计算 | 交付 | 分析结果表齐全 + 计算质检通过 |
| 交付 | 结束 | validate_delivery 通过 + 交付质检通过 |
