# 速查手册

一页掌握常用命令与文件流。

---

## 矩阵解析（模式 A）

```bash
python scripts/parse_matrix.py 原始数据/stats.xlsx --indicator GDP --unit 亿元 -o output/01_原始标准长表.csv
```

## 导出 Excel

```bash
python scripts/export_workbook.py output/
```

## Makefile

```bash
make doctor    # 环境检查
make demo      # 完整演示
make init NAME=my_proj
make all PROJECT=my_proj   # 分析+图+报告+导出+质检
```

```bash
# 1. 检查环境
python scripts/doctor.py

# 2. 初始化项目
python scripts/init_project.py my_project --with-sample

# 3. 分析（已有长表时）
python scripts/check_long_table.py my_project/output/02_清洗后长表.csv
python scripts/analyze_long_table.py my_project/output/02_清洗后长表.csv -o my_project/output/

# 4. 出图
python scripts/plot_delivery.py my_project/output/

# 5. 报告骨架
python scripts/generate_report.py my_project/

# 6. 质检
python scripts/validate_delivery.py my_project/output/ --strict
```

## 文件流

```
原始 xlsx/csv
  → 01_原始标准长表.csv
  → 02_清洗后长表.csv      ← analyze_long_table 输入
  → 03_数据质量检查表.csv
  → 同比分析.csv / 综合指数.csv / 最新年份综合排名.csv
  → 图表/fig01–03
  → 数据分析报告.md
```

---

## 长表必需列

`主体` `年份` `指标名称` `原始值` `清洗值`

推荐另有：`指标方向` `是否纳入综合指数` `单位` `说明`
