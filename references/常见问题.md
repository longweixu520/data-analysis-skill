# 常见问题

## 数据与解析

**Q：表头不在第一行怎么办？**  
先 `probe_excel.py` 抽样，对照 `表结构识别手册.md` 定模式，用 `header=None` 读入再定位表头行。

**Q：多个 sheet 结构不同？**  
一 sheet 一解析函数，最后 `pd.concat`，不要写一个万能解析器。

**Q：合并单元格导致主体为空？**  
读入后对主体列 `ffill()`。

**Q：年份列识别成 2019.0？**  
转 `int`，列名用正则 `20\d{2}` 匹配。

---

## 清洗

**Q：能不能自动删掉异常值？**  
默认不删。先记入 `03_数据质量检查表.csv`，判断是错位还是业务异常再处理。

**Q：缺失太多还能算综合指数吗？**  
按「有效年份筛选」：覆盖不足的年份不硬算，报告说明。

**Q：0.23 和 23% 混了？**  
在术语表统一一种展示方式，清洗值与图表全文一致。

---

## 分析

**Q：排名和综合指数不一致？**  
单指标排名看某一指标；综合指数是多指标加权，二者本来可以不同。

**Q：同比出现 inf？**  
上期为 0 或缺失，应置 NA，检查 `analyze_long_table.py` 产出。

**Q：熵值法权重全差不多？**  
指标区分度低或样本太少，考虑减少纳入指标或换 CRITIC/等权。

---

## 图表

**Q：中文乱码/方框？**  
运行 `plot_delivery.py` 前执行 `doctor.py`；macOS 用 PingFang SC，Windows 用 SimHei。

**Q：主体太多图看不清？**  
排名图只画 Top 15，趋势图只画 Top 5。

## 工具

**Q：`validate_delivery` 警告「无分析结果表」？**  
仅清洗任务可忽略；否则先跑 `analyze_long_table.py`。

**Q：大 CSV 内存爆？**  
`detect_csv.py --recommend` 会建议 polars 或 DuckDB，见 `tools/csv/SKILL.md`。

**Q：脚本 import 失败？**  
在 skill 根目录执行，或 `cd scripts && python analyze_long_table.py ../path/to.csv`。

---

## Agent 使用

**Q：四阶段太慢？**  
先读 `场景路由表.md`，简单任务只跑 1–2 阶段。

**Q：旧 output 要不要信？**  
不信。默认只信用户指定的原始文件，必要时删旧 output 重跑 `run_pipeline.py`。
