# 可视化规范

## 目标

政务/企业交付场景：**清晰、中文、可印刷**。不追求花哨，追求可读与可引用。

---

## 中文字体

```python
import matplotlib.pyplot as plt

plt.rcParams.update({
    "font.sans-serif": ["PingFang SC", "STHeiti", "SimHei", "Microsoft YaHei"],
    "axes.unicode_minus": False,
    "figure.dpi": 150,
    "savefig.dpi": 300,
})
```

绘图前检测可用字体，失败时降级并记录。

---

## 调色板

```python
PALETTE = {
    "primary":   "#0F4D92",
    "secondary": "#3775BA",
    "accent":    "#E8A317",
    "positive":  "#2E9E44",
    "negative":  "#E53935",
    "neutral":   "#767676",
    "grid":      "#E8E8E8",
}
```

- 排名图：单色系渐变或 `primary`
- 趋势图：主体 ≤8 时用区分色，>8 时只高亮 Top5 + 灰色其他
- 涨跌：绿涨红跌（中国惯例）或全文统一一种

---

## 图表类型选择

| 目的 | 推荐 |
|------|------|
| 排名 | 横向条形图 |
| 趋势 | 折线图 |
| 多指标对比 | 热力图 |
| 分维度 | 雷达图（≤5 主体） |
| 权重 | 水平条形图 |
| 分布 | 箱线图/直方图 |

---

## 版式

- 无多余网格线，或仅 y 方向浅灰
- 仅保留左、下 spines
- 数值标签：条形图末端标注，避免重叠
- 图尺寸：单图 `figsize=(10, 6)`，双子图 `(12, 5)`

---

## 导出

每张图输出双格式：

```python
fig.savefig("output/图表/fig01_排名.svg", bbox_inches="tight")
fig.savefig("output/图表/fig01_排名.png", bbox_inches="tight", dpi=300)
```

SVG 设置 `matplotlib.rcParams['svg.fonttype'] = 'none'` 保持文字可编辑。

---

## 防错

- 坐标轴单位与数据一致（亿元 vs 元）
- 年份轴用整数刻度
- 雷达图各轴量纲已标准化到 [0,1]
