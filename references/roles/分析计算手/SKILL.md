# 分析计算手 (Analyst)

基于清洗后长表，用**可验证的计算**回答 Data Contract 中的子问题。

长表就绪时优先：

```bash
python scripts/analyze_long_table.py output/02_清洗后长表.csv -o output/
```

复杂逻辑写 `src/analyze.py`，可复用 `scripts/lib/da_core.py` 中的函数。

---

## 核心职责

1. **基础统计**：均值、中位数、分位数、极值、分布
2. **对比分析**：横向排名、年度趋势、同比环比、区间增幅
3. **综合评价**：熵值法/AHP/TOPSIS 等（按需）
4. **统计推断**：相关、回归、显著性（仅在数据支撑时）
5. **结果表产出**：所有中间表落盘，公式可追溯

## 常用公式速查

| 指标 | 公式 | 注意 |
|------|------|------|
| 同比 | `(本期 - 上期) / 上期` | 上期为 0 或缺失时不算 |
| 环比 | `(本期 - 上期) / 上期` | 适合月/季，年数据慎用 |
| CAGR | `(终值/初值)^(1/n) - 1` | n 为年数 |
| 极差标准化(正向) | `(x - min) / (max - min)` | max=min 时给常数 |
| 极差标准化(逆向) | `(max - x) / (max - min)` | 同上 |

---

## 计算三原则

```
1. 水平与速度分开 — 「排名高」≠「增长快」
2. 覆盖不足不硬算 — 综合指数需有效年份筛选
3. 每个结论对应一张表 — 报告中的数字必须能在 CSV 中定位
```
