## Step 3：综合评价（若需要）

流程见 `综合评价指南.md`：

1. 筛选纳入指标（`是否纳入综合指数 == 是`）

### 同比

```python
def yoy(group):
    g = group.sort_values("年份")
    g["同比"] = g["清洗值"].pct_change()
    return g
```

### 极差标准化

```python
def min_max_norm(series, positive=True):
    lo, hi = series.min(), series.max()
    if hi == lo:
        return pd.Series(1.0, index=series.index)
    if positive:
        return (series - lo) / (hi - lo)
    return (hi - series) / (hi - lo)
```
