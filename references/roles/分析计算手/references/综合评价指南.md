# 综合评价指南

## 何时做综合评价

| 场景 | 建议 |
|------|------|
| 用户明确要求「综合指数」「评价体系」 | 做 |
| 多指标需汇总排名 | 做 |
| 单指标排名即可 | 不做 |
| 指标覆盖年份严重不足 | 先筛有效年份或不做 |

---

## 指标筛选

**纳入**：
- 用户关心的核心指标
- 时间覆盖完整、缺失率低
- 能代表不同维度（避免同质指标堆叠）

**排除**：
- 覆盖年份 < 有效分析窗口的 50%
- 与已纳入指标高度相关（|r|>0.95）的重复指标

---

## 有效年份筛选

对每个候选年份，计算：

```
覆盖率 = 有完整值的主体数 / 总主体数
```

默认：纳入指标在该年均有值的主体占比 ≥ 80%，且纳入指标数 ≥ 计划指标的 80%。

不满足的年份**不计算综合指数**，报告中说明。

---

## 标准化

默认极差标准化（按年截面）：

- 正向：`(x - min) / (max - min)`
- 逆向：`(max - x) / (max - min)`

`max == min` 时该指标该年得分记 1 或 0.5，避免除零。

---

## 权重方法选择

| 方法 | 适用 | 特点 |
|------|------|------|
| **熵值法** | 默认 | 客观，区分度高的指标权重大 |
| 等权 | 指标同等重要 | 简单透明 |
| AHP | 有明确政策导向 | 需成对比较矩阵 |
| CRITIC | 兼顾相关性与波动性 | 客观，计算量适中 |
| 组合赋权 | 主客观结合 | 熵值 + AHP 加权 |

用户未指定时**默认熵值法**。

---

## 熵值法步骤

1. 构建矩阵 `X`（m 个主体 × n 个指标），已标准化到 [0,1]
2. 列归一化：`p_ij = x_ij / sum_i(x_ij)`，遇 0 用 `1e-12` 防 log(0)
3. 熵值：`e_j = -k * sum_i(p_ij * ln(p_ij))`，`k = 1/ln(m)`
4. 差异系数：`d_j = 1 - e_j`
5. 权重：`w_j = d_j / sum(d_j)`

产出 `04_指标权重表.csv`：

| 指标名称 | 熵值 | 差异系数 | 权重 |
|----------|------|----------|------|

---

## 综合指数

```
综合指数_i = sum_j(w_j * x_ij)
```

产出 `综合指数.csv`：`主体, 年份, 综合指数, 排名`

---

## 分维度得分（可选）

若指标有 `指标分类`，可先算维度内等权或熵权，再算维度间权重，产出 `分维度得分.csv` 供雷达图使用。
